# Group 2 - MoM (Movie Match)
> 개인맞춤 영화 추천 및 커뮤니티 서비스

## 프로젝트 소개
MoM(Movie Match)은 사용자의 취향, OTT 구독 정보, 활동 데이터를 기반으로 영화를 추천하고,  
다른 사용자들과 영화 취향과 감상을 공유할 수 있도록 설계한 영화 추천 커뮤니티 서비스입니다.

사용자의 선호 장르, 영화, 배우, 감독, 구독 중인 OTT 서비스 등을 반영하여  
더 현실적이고 개인화된 추천을 제공하는 것을 목표로 합니다.

---

## 서비스 개요
MoM은 **개인 맞춤 추천**, **OTT 서비스 연결**, **함께보기 추천**, **커뮤니티 기능**을 중심으로 구성된 서비스입니다.

사용자는 회원가입 시 입력한 취향 정보와 OTT 구독 정보를 바탕으로 초기 추천을 받을 수 있으며,  
이후 시청 기록, 좋아요, 북마크, 평점 등의 활동 데이터가 누적될수록  
추천 결과가 점점 더 개인화되도록 설계합니다.

또한 혼자 영화를 찾는 상황뿐 아니라,  
가족이나 친구와 함께 볼 영화를 정해야 하는 상황까지 고려하여  
여러 사용자의 취향을 함께 반영하는 추천 기능을 제공하는 것을 목표로 합니다.

---

## 기획 배경
- OTT 서비스와 콘텐츠가 많아 무엇을 볼지 선택하기 어려움
- 단순 인기순 추천만으로는 개인 취향을 충분히 반영하기 어려움
- 원하는 영화가 어떤 OTT에서 제공되는지 따로 검색해야 하는 불편함이 존재함
- 가족, 친구와 함께 볼 영화를 정할 때 공통 취향을 만족시키기 어려움
- 영화 취향과 감상을 공유할 수 있는 커뮤니티/SNS 기능 수요가 존재함

---

## 핵심 기능

### 1. 개인 맞춤 추천
- 선호 장르, 선호 영화, 선호 배우, 선호 감독 정보를 반영한 영화 추천
- 사용자 기본 정보와 활동 데이터(시청 기록, 좋아요, 평점, 북마크)를 바탕으로 추천 고도화
- 단순 인기순이 아닌, 사용자 취향 중심의 개인화 추천 제공

### 2. OTT 서비스 연결
- 영화 상세 페이지에서 해당 영화를 제공하는 OTT 서비스 확인
- 사용자가 구독 중인 OTT를 기준으로 현실적인 추천 제공
- 보고 싶은 영화를 찾은 뒤 실제 시청 가능한 플랫폼으로 바로 이동 가능

### 3. 함께보기 추천
- 가족, 친구 등 여러 사용자의 취향을 함께 반영한 영화 추천
- 공통으로 만족할 가능성이 높은 영화를 탐색할 수 있도록 지원
- 혼자 보는 상황뿐 아니라 함께 보는 상황까지 고려한 추천 기능 제공

### 4. 커뮤니티 및 공유 기능
- 영화에 대한 감상 공유, 댓글, 토론 기능
- 사용자 간 영화 취향을 기반으로 한 소통 지원
- 단순 추천 서비스에서 나아가 영화 취향을 공유하는 커뮤니티 지향

---

## 차별화 포인트
- **개인 취향 기반 추천**  
  단순 인기순 추천이 아니라, 선호 장르·배우·감독·영화와 활동 로그를 반영한 추천 제공

- **OTT 구독 정보 반영**  
  사용자가 실제로 이용 중인 OTT 서비스를 기준으로 추천하여 실사용성을 높임

- **함께보기 추천 기능**  
  여러 사용자의 취향을 함께 고려해 같이 볼 영화를 추천하는 기능 제공

- **커뮤니티 + 공유 기능 결합**  
  영화 정보 조회에 그치지 않고, 감상과 취향을 나눌 수 있는 커뮤니티형 서비스 지향

---

## 구현 방향
- **Spring Boot 기반 백엔드**
- **외부 영화 API를 통한 영화 메타데이터 수집**
- **사용자 활동 로그 저장 및 추천에 반영**
- **DB 관계 엔티티 구성**
- **추천 알고리즘 적용**
  - 콘텐츠 기반 추천
  - 사용자 기반 추천
  - 하이브리드 추천
- **AI를 활용한 분위기/감성 정보 보강 확장 가능**

## 기대 효과
- 개인 취향에 맞는 영화 추천 제공
- 영화 탐색 과정의 편의성 향상
- 보고 싶은 영화의 OTT 제공 여부를 쉽게 확인 가능
- 가족, 친구와 함께 볼 영화 선택 문제 해결
- 커뮤니티 기반 영화 소통 활성화
